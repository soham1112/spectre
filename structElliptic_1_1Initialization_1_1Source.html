Detailed Description

template<typename Metavariables>
struct Elliptic::Initialization::Source< Metavariables >

Computes the sources of the elliptic equations and adds them to the DataBox.

Note
Currently the sources are always retrieved from an analytic solution.

With:

  • sources_tag = db::add_tag_prefix<Tags::Source, system::fields_tag>

Uses:

  • Metavariables:
    • analytic_solution_tag
  • ConstGlobalCache:
    • analytic_solution_tag
  • System:
    • volume_dim
    • fields_tag
  • DataBox:
    • Tags::Mesh<volume_dim>
    • Tags::Coordinates<volume_dim, Frame::Inertial>

DataBox:

  • Adds:
    • sources_tag
