Detailed Description

template<typename Metavariables>
class ActionTesting::MockRuntimeSystem< Metavariables >

A class that mocks the infrastructure needed to run actions. It simulates message passing using the inbox infrastructure and handles most of the arguments to the apply and is_ready action methods.
