Detailed Description

template<typename Metavariables>
struct Elliptic::Initialization::LinearSolver< Metavariables >

Initializes the DataBox tag related to the linear solver.

Uses the analytic solution to compute the linear solver source. Assumes the system fields are initialized to zero (see Elliptic::Initialization::System) so we don't have to apply the operator to the initial guess.

With:

  • sources_tag = db::add_tag_prefix<Tags::Source, system::fields_tag>

Uses:

  • Metavariables:
    • linear_solver
  • System:
    • fields_tag
  • DataBox:
    • sources_tag

DataBox:

  • Adds:
    • All items in linear_solver::tags
